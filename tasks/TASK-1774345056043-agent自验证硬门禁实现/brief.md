# agent自验证硬门禁实现

## 目标

- 实现仓库级可复用的 agent 自验证硬门禁，使用熟悉工具作为检查本体，在每个小步提交前通过全量 gate 阻断结构、逻辑、交互与工作流错误。

## 范围

### In

- 新增 verify:gate 与四类子 gate 入口
- 实现结构/逻辑/交互/工作流四类门禁
- 失败时输出机器可判定的分类与证据
- 将门禁接入仓库文档与后续 agent 工作流

### Out

- 不接入真实外部依赖验证
- 不生成冗长成功汇总报告

## 验收标准

- 每个小步提交前可运行同一个全量 gate
- 任一子 gate 失败都会阻断提交
- 失败结果能给出分类与证据
